## 生活场景设计（Basel本地化）

### Stage 1: BASIC_PROB - 基础概率（日常决策）

#### 场景1: Basel公交准点率
**情境**: 
- 你每天乘坐Basel 8路公交车上学
- 过去20天中，公交准时到达16次
- 问：明天公交准时的概率是多少？

**知识点**: 频率估计概率
**计算**: P = 16/20 = 0.8 = 80%

**难度递进**:
- BASIC: 简单分数（16/20）
- CORE: 需要化简（12/15 = 4/5）
- ADVANCED: 多组数据比较（哪条线路更准时？）
- ELITE: 条件概率引入（雨天vs晴天的准点率）

#### 场景2: Basel天气预报
**情境**:
- Basel气象站统计：过去30天中下雨12天
- 问：本周末下雨的概率？

**应用**: 决定是否计划户外活动（莱茵河游泳、动物园）

#### 场景3: 学校食堂菜单
**情境**:
- 食堂每周5天，有3天提供披萨
- 随机选一天去食堂，吃到披萨的概率？

**生活关联**: 帮助学生理解"随机"和"概率"的实际意义

### Stage 2: LOTTERY - 简单博彩应用（Swiss Lotto简化版）

#### 场景1: Basel学校抽奖活动
**情境**:
- 学校慈善活动，售出100张彩票
- 你买了3张
- 问：你中奖的概率？

**知识点**: 
- 单次中奖概率 = 3/100 = 3%
- 不中奖概率 = 97/100 = 97%

**教育意义**: 理解"小概率事件"

#### 场景2: Swiss Lotto简化版（6选3）
**情境**:
- 简化版Swiss Lotto：从6个数字中选3个
- 总共有多少种组合？（不教组合公式，用列举）
- 你选的一组中奖概率是多少？

**计算**:
- 总组合数：C(6,3) = 20（可以列举出来）
- 中奖概率：1/20 = 5%

**对比真实Swiss Lotto**:
- 真实版：从42个数字中选6个
- 中奖概率：约1/5,245,786（0.000019%）
- **教育意义**: 理解为什么"不要沉迷博彩"

#### 场景3: 骰子游戏（Basel Fasnacht嘉年华）
**情境**:
- Basel狂欢节游戏摊位
- 规则：掷2个骰子，和为7或11赢
- 问：获胜概率？

**计算**:
- 和为7的方式：(1,6), (2,5), (3,4), (4,3), (5,2), (6,1) = 6种
- 和为11的方式：(5,6), (6,5) = 2种
- 总共36种可能
- P(赢) = 8/36 = 2/9 ≈ 22.2%

**教育意义**: 理解"庄家优势"

### Stage 3: COMBINED - 组合事件（实际应用）

#### 场景1: Basel足球比赛预测
**情境**:
- FC Basel主场比赛
- 历史数据：主场胜率60%，客场胜率30%
- 本周主场+客场两场比赛
- 问：两场都赢的概率？

**计算**: P(都赢) = 0.6 × 0.3 = 0.18 = 18%

#### 场景2: 连续公交准点
**情境**:
- 早上乘8路公交（准点率80%）
- 晚上乘15路公交（准点率70%）
- 问：两趟都准点的概率？

**计算**: P = 0.8 × 0.7 = 0.56 = 56%

#### 场景3: Novartis质量控制（职业体验）
**情境**:
- 你在Novartis实习，检查药品质量
- 每批次抽检5个样品
- 合格率95%
- 问：5个全部合格的概率？

**计算**: P = 0.95^5 ≈ 0.774 = 77.4%

**职业教育**: 了解制药行业的质量标准

### Stage 4: DATA_ANALYSIS - 数据统计（实用技能）

#### 场景1: 班级考试成绩分析
**情境**:
- 数学考试成绩分布
- 计算：平均分、中位数、众数
- 绘制：柱状图

**数据示例**:
```
分数段    人数
90-100    5
80-89     12
70-79     8
60-69     3
<60       2
```

**问题**:
1. 平均分是多少？
2. 中位数在哪个区间？
3. 哪个分数段人数最多？

#### 场景2: Basel气温统计
**情境**:
- 记录一周气温
- 计算平均温度
- 判断哪天最热/最冷

**实用性**: 理解天气预报中的"平均温度"

#### 场景3: 零花钱支出分析
**情境**:
- 记录一个月的零花钱支出
- 分类：食物、交通、娱乐、储蓄
- 绘制饼图
- 分析：哪类支出最多？

**生活技能**: 培养理财意识

## 难度分级

### BASIC（基础）
- 简单分数概率（1/6, 1/2）
- 直接频率计算
- 单一事件

### CORE（核心）
- 需要化简的分数
- 百分比转换
- 简单的"或"事件

### ADVANCED（进阶）
- 两个独立事件的"与"
- 频率表分析
- 简单数据统计

### ELITE（精英）
- 三个事件组合
- 条件概率引入
- 复杂数据分析
