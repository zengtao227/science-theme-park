# GM3.01 新设计总结 - 概念深度递进

## ✅ 已完成的修改

### 1. 重新设计了所有问题数据

#### BASIC_PROB 阶段 - 真正的概念递进

**BASIC 难度**：直接样本空间（6-10个元素）
- 单个骰子（6个面）
- 单个硬币（2个面）
- 8格转盘
- **概念**：样本空间直接可见

**CORE 难度**：理解组合和"等可能性"（36-52个元素）
- 两个骰子的和为7（需要理解36种组合）
- 两个骰子的和为10
- 两个骰子的和>7
- 一副牌中的一个花色
- 一副牌中的荣誉牌（A,K,Q,J）
- **概念**：需要理解不同排列是不同结果

**ADVANCED 难度**：条件概率（隐含）
- 骰子点数>3的条件下，是偶数的概率（2/3而不是1/2）
- 骰子是3的倍数的概率
- 黑桃牌中是人头牌的概率
- 不是人头牌也不是A的牌
- 人头牌中是K的概率
- **概念**：条件改变了样本空间

**ELITE 难度**：复合事件
- 两个骰子至少有一个6（需要用补集：1 - P(都不是6)）
- 两个骰子的和不是2也不是12
- 两个骰子至少有一个偶数
- 一张牌是A或K
- 一张牌是红色或人头牌
- **概念**：需要分解事件，使用加法/乘法原理

#### BINOMIAL 阶段 - 真正的概念递进

**BASIC 难度**：理解二项分布基本概念（n=3-4, p=0.5）
- 3次硬币，恰好2次正面
- 4次硬币，恰好2次正面
- 3次硬币，全部正面
- 4次硬币，全部反面
- **概念**：理解"恰好k次"的含义

**CORE 难度**：理解二项系数C(n,k)的意义（n=5-7, p=0.5）
- 5次试验，恰好3次成功（C(5,3)=10种方式）
- 6次试验，恰好3次成功（C(6,3)=20种方式）
- **概念**：理解为什么要乘以C(n,k)

**ADVANCED 难度**：非对称概率（p≠0.5）
- n=5, k=3, p=0.6（偏态分布）
- n=6, k=2, p=0.3（低概率）
- n=8, k=6, p=0.7（高概率）
- **概念**：理解偏态分布

**ELITE 难度**：累积概率
- "至少3次成功"（需要计算P(X≥3) = P(X=3) + P(X=4) + P(X=5)）
- "至多4次成功"（需要计算P(X≤4)）
- "超过一半成功"
- **概念**：累积概率的计算

#### CONDITIONAL 阶段 - 真正的概念递进

**BASIC 难度**：理解公式P(A|B) = P(A∩B) / P(B)
- 直接给出三个概率值，套用公式
- **概念**：理解条件概率的定义

**CORE 难度**：从问题中提取条件
- "已知是红色牌，求是红心的概率"（需要识别B="红色"，A="红心"）
- "已知是偶数，求是6的概率"
- **概念**：从文字描述中识别条件和事件

**ADVANCED 难度**：贝叶斯思维
- "测试阳性，真正患病的概率"（P(病|阳性) ≠ P(阳性|病)）
- "质量警报，真正有缺陷的概率"
- **概念**：理解P(A|B)和P(B|A)的区别

**ELITE 难度**：独立性判断
- 判断两个事件是否独立（P(A|B) = P(A)?）
- 多重条件概率
- **概念**：理解独立性的定义

## 🎯 学习目标

通过4个难度级别，学生应该能够：

**BASIC**: 
- 理解基本概率定义 P(E) = favorable/total
- 能够数出样本空间

**CORE**: 
- 理解样本空间的构造（组合）
- 理解"等可能性"的含义

**ADVANCED**: 
- 理解条件如何改变样本空间
- 能够识别隐含的条件概率

**ELITE**: 
- 能够分解复杂问题
- 使用补集、加法原理、乘法原理
- 理解独立性和贝叶斯思维

## 💡 设计理念

**核心思想**：难度不是数量，而是思维方式的转变

- BASIC → CORE: 从"看到"到"构造"
- CORE → ADVANCED: 从"全部"到"部分"（条件）
- ADVANCED → ELITE: 从"单一"到"复合"（分解）

这才是真正的学习曲线！
