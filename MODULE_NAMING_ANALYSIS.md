# 模块命名分析和统一方案

**分析日期**: 2026-02-14

---

## 🔍 当前命名问题分析

### 问题1: 命名不一致

**Gymnasium数学模块的命名混乱**:
- ✅ 有些用 `G` 前缀: G1.01, G2.01, G3.01, G4.01, G5.01
- ✅ 有些用 `GM` 前缀: GM1.01, GM2.01, GM3.01, GM4.01, GM5.01
- ❓ 还有 `GMS` 前缀: GMS1.01

**实际情况**:
- 在代码中: 文件夹名都是 `gm1-01`, `gm2-01` 等 (用GM)
- 在i18n中: 有些显示为 `G1.01`, 有些显示为 `GM1.01`
- 在CURRICULUM_PLAN中: 显示为 `G1.01`, `G2.01` 等

---

## 📚 GM1.01 vs GMS1.01 的区别

### GM1.01 - CALCULUS INTRO (微积分初步)
**内容**: 
- 导数基础 (Derivatives)
- 幂法则 (Power Rule)
- 因子法则 (Factor Rule)
- 和法则 (Sum Rule)
- 乘积法则 (Product Rule)
- 商法则 (Quotient Rule)
- 链式法则 (Chain Rule)

**教学目标**: 
- 理解导数的几何意义(切线斜率)
- 掌握基本求导法则
- 计算函数在某点的导数

**适用阶段**: Gymnasium (高中) 数学核心课程

**类型**: Quest系统 + 可视化

---

### GMS1.01 - FRACTAL EXPLORER (分形探索者)
**内容**:
- Mandelbrot集合
- 复数迭代
- 分形几何
- 自相似性

**教学目标**:
- 探索分形的美学
- 理解复数迭代
- 可视化数学之美

**适用阶段**: Gymnasium Special/Supplementary (高中特殊/补充内容)

**类型**: 纯可视化工具,没有Quest系统

## 🔧 命名统一方案

### 方案A: 全部使用GM前缀 (推荐)

**优点**:
- 与代码文件夹名一致 (gm1-01, gm2-01)
- 清晰表示 Gymnasium Mathematics
- 易于理解和维护

**实施**:
```
当前              →  统一后
G1.01            →  GM1.01 (Calculus)
G2.01            →  GM2.01 (Vectors)
G3.01            →  GM3.01 (Probability)
G4.01            →  GM4.01 (Complex Numbers)
G5.01            →  GM5.01 (Matrix Geometry)
GMS1.01          →  GM6.01 (Fractal Explorer) 或删除
```

### 方案B: 保留GMS作为特殊标记

**优点**:
- 区分核心课程和补充内容
- GMS = Gymnasium Math Special

**缺点**:
- 增加复杂度
- 不符合其他学科的命名规范

## 💡 关于GMS1.01 (分形探索者) 的建议

### 问题分析:

1. **不符合模块标准**:
   - 没有Quest系统
   - 没有难度分级
   - 没有巴塞尔场景
   - 只是一个可视化工具

2. **课程定位不明确**:
   - 分形不是瑞士高中数学的核心内容
   - 更像是一个"展示"而非"教学"模块
   - 与其他模块的教学深度不匹配

3. **命名混乱**:
   - GMS前缀在整个系统中是唯一的
   - 破坏了命名一致性

### 建议方案:

#### 选项1: 删除GMS1.01 (推荐)
**理由**:
- 不符合CHAMBER_MODULE_STANDARDS
- 不是核心课程内容
- 维护成本高,教学价值低
- 可以用其他方式展示分形(如在GM4.01复数模块中)

#### 选项2: 重新设计为GM6.01
**如果保留,需要**:
- 添加Quest系统
- 添加难度分级
- 添加巴塞尔场景
- 设计教学目标(如:理解递归、迭代、复数运算)
- 符合CHAMBER_MODULE_STANDARDS

#### 选项3: 移到"展示区"
- 不作为正式模块
- 作为"数学之美"的展示
- 放在单独的展示页面

## 🎓 瑞士高中数学课程内容

根据Basel课程标准,Gymnasium数学核心内容:

**必修内容**:
1. ✅ 微积分 (Calculus) - GM1.01
2. ✅ 向量几何 (Vector Geometry) - GM2.01
3. ✅ 概率统计 (Probability) - GM3.01
4. ✅ 复数 (Complex Numbers) - GM4.01
5. ✅ 矩阵/线性代数 (Linear Algebra) - GM5.01

**可选/拓展内容**:
- 数论 (Number Theory)
- 组合数学 (Combinatorics)
- 微分方程 (Differential Equations)
- ❓ 分形几何 (Fractals) - 不是标准课程

**结论**: 分形不是瑞士高中数学的标准内容,更适合作为"数学之美"的展示,而非正式教学模块。
