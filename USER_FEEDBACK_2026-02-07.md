# 用户反馈总结 - 2026年2月7日

## 全局问题（影响所有模块）

### 问题 1: 模块图标缺失
**现状**: 主页上的模块卡片没有显示概念图标
**期望**: 每个模块应该有一个相关的图标（如三角形、立方体、函数图等）
**影响范围**: 所有53个模块

### 问题 2: 左右分界线无法拖动
**现状**: 进入模块后，左侧内容区和右侧可视化区的分界线固定
**期望**: 可以拖动分界线调整左右区域大小，以便放大右侧可视化区域
**影响范围**: 所有模块页面

## SM1.01 面积与体积模块具体问题

### 问题 3: 立方体无法旋转
**现状**: 体积部分的立方体是静态的，无法看清每条边的长度
**期望**: 立方体应该可以旋转，方便查看所有边长（都是4）
**位置**: SM1.01 体积计算部分

### 问题 4: 缺少任务背景说明
**现状**: 任务直接显示"展开蓝图以购买准确面积的地毯"，没有上下文
**期望**: 添加情境说明，例如：
```
有一个人需要在家里更换地毯。他测量了房间的尺寸，
现在需要计算需要购买多少面积的地毯。请帮助他解决这个问题。

当前任务：展开蓝图以购买准确面积的地毯
```
**影响**: 所有任务描述（回收员、疏算者等）

## SM2.01 二项式工厂模块具体问题

### 问题 5: 自动旋转干扰观察
**现状**: 立方体不停自动旋转，难以仔细观察
**期望**: 
- 默认不自动旋转
- 只有用户拖动时才旋转
- 添加"Reset"按钮恢复到初始视角

### 问题 6: (a+b)³ 爆炸动画问题
**子问题 6.1**: 爆炸后方块方向不一致，难以理解
**子问题 6.2**: 爆炸展开时间太短（约1-2秒），来不及观察
**子问题 6.3**: 爆炸后屏幕仍然太小，需要更大的可视化区域
**子问题 6.4**: 无法控制动画，自动循环播放

**期望**: 
- 爆炸后方块应该对齐，方便理解结构
- 延长爆炸展开状态的停留时间（至少5-10秒）
- 添加动画控制按钮：
  - "展开" - 触发爆炸动画
  - "收起" - 恢复到立方体状态
  - "Reset" - 重置视角
  - "暂停/播放" - 控制自动循环
- 配合可拖动分界线，让用户自行调整可视化区域大小
