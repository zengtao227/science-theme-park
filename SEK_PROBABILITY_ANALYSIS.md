# Sekundarschule概率教学分析

## 问题
初中（Sekundarschule）阶段是否学习概率知识？应该放在哪个模块？

## 现状分析

### 1. 当前模块情况

#### GM3.01 - Probability Vault (Gymnasium)
- **级别**: Gymnasium（高中）
- **内容**: 
  - 基础概率 (Basic Probability)
  - 二项分布 (Binomial Distribution)
  - 条件概率 (Conditional Probability)
  - 独立性检验 (Independence Testing)
- **应用场景**: 
  - Novartis质量控制
  - Swiss Lotto彩票分析
  - Basler保险风险评估

#### SM1.02 - 当前实际内容
- **名称混淆**: README中标注为"数据与概率"
- **实际内容**: 泰勒斯定理（相似三角形测量）
- **应用场景**: 钟楼高度测量
- **状态**: 内容与名称不符

### 2. 瑞士课程标准 (Lehrplan 21)

根据Lehrplan 21框架，概率教学分布：

#### Zyklus 2 (小学高年级, 5-6年级)
- **内容**: 非常基础的概率概念
- **例子**: 
  - 抛硬币（正面/反面）
  - 掷骰子（1-6）
  - 简单的"可能性"概念

#### Zyklus 3 (初中, 7-9年级 / Sekundarschule)
- **内容**: 系统性概率学习
- **知识点**:
  - 概率的定义和计算 P(E) = favorable/total
  - 简单事件的概率
  - 频率与概率的关系
  - 基础统计图表（柱状图、饼图）
  - 数据收集与分析
- **深度**: 不涉及二项分布、条件概率等高级内容

#### Gymnasium (高中)
- **内容**: 高级概率论
- **知识点**:
  - 二项分布
  - 条件概率
  - 贝叶斯定理
  - 独立性
  - 正态分布

## 建议方案

### 方案1: 创建新模块 SM3.04 - 基础概率与统计

**理由**:
- SM3系列是Sek最高级别（对应8-9年级）
- 概率通常在初中后期教授
- 与SM3.01（二次方程）、SM3.02（三角函数）、SM3.03（指数增长）难度相当

**内容设计**:
```
SM3.04 - Probability & Statistics Basics
├─ Stage 1: BASIC_PROBABILITY (基础概率)
│  ├─ 抛硬币、掷骰子
│  ├─ P(E) = favorable/total
│  └─ 简单事件概率计算
│
├─ Stage 2: DATA_ANALYSIS (数据分析)
│  ├─ 频率表
│  ├─ 柱状图、饼图
│  └─ 平均值、中位数
│
└─ Stage 3: COMBINED_EVENTS (组合事件)
   ├─ 两个事件的概率
   ├─ "与"和"或"的概率
   └─ 简单的树状图
```

**应用场景**:
- Swiss Lotto基础玩法（不涉及复杂计算）
- Basel天气预报概率
- 学校考试成绩统计
- 公交车准点率分析

**难度对比**:
| 模块 | 概率深度 | 数学工具 |
|------|---------|---------|
| SM3.04 | 基础概率 | 分数、百分比、简单乘法 |
| GM3.01 | 高级概率 | 组合数学、二项式定理、条件概率公式 |

### 方案2: 修正SM1.02模块

**当前问题**:
- README标注为"数据与概率"
- 实际内容是"泰勒斯定理"（相似三角形）
- 名称与内容不符

**修正选项**:

#### 选项A: 保持SM1.02为泰勒斯定理
- 将README中的描述改为"相似三角形与测量"
- 删除"数据与概率"的描述
- 在SM3.04创建真正的概率模块

#### 选项B: 将SM1.02改造为概率模块
- 将当前泰勒斯内容移到SM1.02-similarity
- 将SM1.02改造为基础概率与统计
- 但SM1系列通常是最基础内容（7年级），概率可能太早

## 推荐方案

### ✅ 推荐: 方案1 + 选项A

1. **创建SM3.04** - 基础概率与统计
   - 适合8-9年级学生
   - 内容符合Lehrplan 21的Zyklus 3要求
   - 与GM3.01形成递进关系

2. **修正SM1.02命名**
   - 保持当前泰勒斯定理内容
   - 更新README描述为"相似三角形与测量"
   - 避免混淆

3. **模块层次结构**
```
初中 (Sekundarschule):
  SM3.04 - 基础概率与统计
  ├─ 简单概率计算
  ├─ 数据统计
  └─ 基础图表分析

高中 (Gymnasium):
  GM3.01 - 高级概率论
  ├─ 二项分布
  ├─ 条件概率
```
