# Basel（Basel-Stadt / Basel-Landschaft）Sek I（约 8 年级）数学资料入口

本文件用于长期维护 MG05 / MG06（以及后续数学主题）设计与题目编制时的“权威/常用/可复查”资料入口，便于持续迭代时回溯依据。

  - https://www.lehrplan21.ch/kanton-basel-stadt
  - 用途：确认 Basel-Stadt 的官方课程框架采用 Lehrplan 21，并以此作为能力目标（Kompetenzen）上位依据。
- Lehrplan 21（Basel-Stadt 镜像入口）
  - https://bs.lehrplan.ch/
  - 用途：Lehrplan 21 的州级入口（便于定位到 Mathematik / Zyklus 3 的能力描述）。
- Lehrplan 21（Mathematik / Zyklus 3 总览）
  - https://v-fe.lehrplan.ch/index.php?code=e%7C5%7C3
  - 用途：Zyklus 3 数学“能力—内容—活动（Handlungsaspekte）”总体结构，是我们决定“交互要练过程还是只练答案”的核心依据。

## 4) 典型练习/复习资料（用于“题型与写法对齐”）

- Kantonale Fachschaft Mathematik（LU）Repetitionsaufgaben：Termumformungen（含 Faktorisieren 的关键恒等式写法）
  - https://mse.lu.ch/-/media/KSReussbuehl/Dokumente/Ausbildung/Fachschaften/Mathematik/Repetitionsprogramme/Repetitionsaufgaben_Termumformungen_1.pdf?rev=0f7e9b55d15649be81b604b26a404e7b
  - 用途：我们在 MG06 “因式分解/解方程”里应显式呈现的关键过程提示，例如：
    - 若可写成 \(x^2+(a+b)x+ab\)，则可因式分解为 \((x+a)(x+b)\)
    - 这种写法能直接指导“提示层级”与“槽位输入（A+B / AB）”的设计。

## 5) 如何使用这些资料（落地规则）

- 题目范围（Scope）
  - 先用 Lehrplan 21（Zyklus 3）定“能力目标”，再用 Baselland 的 Stoffinhalte/教材章节定“年级进度”。
- 难度分层（BASIC/CORE/ADVANCED/ELITE）
  - 用“年级进度”决定是否允许引入扩展内容（Erweiterung），用“能力目标”决定是否要求过程与表征。
- 交互设计
  - 优先用“槽位输入/分步验证/可视化构造”来体现 Lehrplan 21 强调的 Mathematisieren und Darstellen，而非单纯刷答案。
